# Boolean Logic

## Summary
- Practical branching and state-combination rules using `bool`.

## Syntax
```cpp
bool canAccess = canLogin(true, true, 2);
bool lockNow = shouldLockAccount(6, false);
```

## Complexity
- Boolean checks and comparisons are `O(1)`.

## Memory
- `bool` is a value type with trivial storage.
- Local boolean expressions are stack values.
