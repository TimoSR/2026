# Normal Rust Project Example

This is the intended Rust Plus workflow.

You write a normal Cargo project. The only difference is that source files can be `.rp` instead of `.rs`.

```text
src/
├── main.rp
└── account.rp
```

`build.rs` runs before Rust compilation and generates sibling `.rs` files:

```text
src/
├── main.rp
├── main.rs      # generated by rustplus
├── account.rp
└── account.rs   # generated by rustplus
```

Run it:

```bash
cargo run
```

This example also shows Rust Plus interface construction in `main.rp`:

```rust
let mut account: IAccount = Account::Heap("account-1");
```

Generated Rust:

```rust
let mut account: Box<dyn IAccount> = Box::new(Account::new("account-1".to_string()));
```

`Cargo.toml` contains an explicit binary target so `build.rs` can generate `src/main.rs` from `src/main.rp` before Rust compiles the binary:

```toml
[[bin]]
name = "normal-rust-project"
path = "src/main.rs"
```
