# cout_style_output

`cout_style_output.cpp` shows stream-style output (`<<`) with explicit composition.

Use this when:
- you are on C++17/C++20 toolchains without `<print>`
- you want familiar stream operators for custom types

See also:
- `println_style_cpp23.cpp` for modern `std::print` style
- `println_vs_cout.md` for side-by-side comparison
