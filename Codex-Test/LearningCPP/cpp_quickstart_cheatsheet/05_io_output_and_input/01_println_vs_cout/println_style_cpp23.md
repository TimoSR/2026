# println_style_cpp23

`println_style_cpp23.cpp` demonstrates modern formatted output with:
- `std::format(...)`
- `std::print(...)`
- `std::println(...)`

Notes:
- Requires a C++23-capable standard library implementation.
- The test file uses `println_style_cpp23.cpp23.test.cpp` naming to signal C++23 compilation in this repo's test runner.

See also:
- `cout_style_output.cpp` for stream-based style
- `println_vs_cout.md` for comparison guidance
