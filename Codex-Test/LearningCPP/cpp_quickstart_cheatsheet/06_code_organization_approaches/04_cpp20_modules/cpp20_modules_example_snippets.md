# cpp20_modules_example_snippets

`cpp20_modules_example_snippets.cpp` contains lightweight module-related snippets that compile without full module build setup.

Use it for:
- syntax familiarity
- quick demonstrations in environments where full module tooling is unstable

For real module build/use flow, see:
- `bank_account_module.ixx`
- `bank_account_module_usage.cpp`
- `cpp20_modules.md`
